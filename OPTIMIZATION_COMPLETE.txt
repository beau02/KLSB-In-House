Redis installation is OPTIONAL. App works without it using in-memory caching.

If you want to use Redis:

1. Install Redis:
   Windows: https://github.com/microsoftarchive/redis/releases
   macOS: brew install redis
   Linux: apt-get install redis-server

2. Create .env in backend/:
   ENABLE_REDIS=true
   REDIS_HOST=localhost
   REDIS_PORT=6379
   REDIS_DB=0

3. Start Redis:
   redis-server

4. Restart backend:
   npm run dev

5. Verify connection:
   redis-cli ping
   # Should return: PONG
